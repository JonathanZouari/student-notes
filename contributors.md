## כיצד להצטרף?

נשמח לקבל תורמים נוספים! כך תוכלו להצטרף לפרויקט:

1. **Fork** — לחצו על כפתור Fork בדף הרפוזיטורי
2. **Clone** — הורידו את ה-Fork שלכם למחשב:
   ```bash
   git clone https://github.com/<your-username>/student-notes.git
   ```
3. **ענף חדש** — צרו ענף עבודה נפרד:
   ```bash
   git checkout -b feature/your-contribution
   ```
4. **תרמו** — הוסיפו סיכומים, תקנו שגיאות, שפרו תיעוד
5. **Commit ו-Push** — שמרו ודחפו את השינויים:
   ```bash
   git add .
   git commit -m "Add: תיאור קצר של השינוי"
   git push origin feature/your-contribution
   ```
6. **Pull Request** — פתחו PR ברפוזיטורי המקורי עם תיאור ברור של מה הוספתם

> **טיפ:** לפני שמתחילים, כדאי לפתוח Issue ולתאר את הכוונה — כך נמנעים מכפל עבודה.
