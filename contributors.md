## כיצד להצטרף?

אנחנו שמחים לקבל תורמים חדשים! כך תוכלו להצטרף:

1. **Fork** — צרו עותק של המאגר לחשבון שלכם
2. **Branch** — פתחו ענף חדש עבור השינוי שלכם (`git checkout -b feature/your-feature`)
3. **Commit** — בצעו את השינויים והוסיפו הודעת commit ברורה
4. **Pull Request** — שלחו PR לענף `main` עם תיאור מה שינה ולמה
5. **Review** — המתינו לסקירת הקוד ולאישור

> לפני הכל, מומלץ לפתוח **Issue** ולתאר את הרעיון שלכם — כך נוכל לתאם ולהימנע מכפילויות.
