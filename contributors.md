## 🤝 כיצד להצטרף?

אנו מזמינים כל אחד להתרום לפרויקט! להלן קווים מנחים לתרומה:

### שלבים להתחלה:
1. **Fork את הריפוזיטוריום** - יוצרים עותק אישי של הפרויקט
2. **צור ענף חדש** - `git checkout -b feature/your-contribution`
3. **בצע שינויים** - תרום תוכן, קוד או שיפורים
4. **Commit וPush** - `git push origin feature/your-contribution`
5. **יוצר Pull Request** - פתח PR עם תיאור ברור של התרומה

### סוגי תרומות מוכנות:
- 📝 **תוכן**: כתיבת מדריכים, מאמרים ופתרונות
- 🐛 **תיקון באגים**: זיהוי ותיקון בעיות קיימות
- ✨ **תכונות חדשות**: הצעת וביצוע פונקציונליות חדשה
- 📚 **תיקיות**: שיפור התיעוד וההנחיות
- 🔍 **ביקורת**: ביקורת כללית על Pull Requests וקוד
