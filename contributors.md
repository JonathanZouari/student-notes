## 🤝 How to Contribute

We welcome contributions from students and educators alike! Whether you're fixing a typo or adding a full study guide, every contribution matters.

### Getting Started

Follow these steps to submit your contribution:

1. **Fork** the repository  
   Click the **Fork** button at the top-right of this page to create your own copy of the project.

2. **Create a Branch**  
   Create a descriptive branch for your changes:
   ```bash
   git checkout -b feature/your-topic-name
   ```

3. **Make Your Changes & Commit**  
   Write clear, meaningful commit messages:
   ```bash
   git add .
   git commit -m "Add: study notes for Chapter 3 - Data Structures"
   ```

4. **Push & Open a Pull Request**  
   Push your branch and open a Pull Request against `main`:
   ```bash
   git push origin feature/your-topic-name
   ```
   Then go to GitHub, click **Compare & pull request**, fill in the description, and submit.

### ✅ Contribution Guidelines

- Keep notes **clear and well-structured**
- Use **proper Markdown formatting**
- One topic per Pull Request
- Be respectful and constructive in reviews
