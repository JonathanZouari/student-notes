## כיצד להצטרף?

אנחנו שמחים לקבל תורמים חדשים! כך תוכל/י להשתתף:

1. **Fork** את ה-repository דרך כפתור ה-Fork בפינה הימנית העליונה
2. **Clone** את ה-fork שלך למחשב:
   ```bash
   git clone https://github.com/<your-username>/student-notes.git
   ```
3. **צור branch חדש** עבור השינוי שלך:
   ```bash
   git checkout -b feature/my-contribution
   ```
4. **הוסף את התוכן שלך** — סיכום חדש, תיקון שגיאה, שיפור תיעוד
5. **Commit ו-Push**:
   ```bash
   git add .
   git commit -m "Add: <תיאור קצר של השינוי>"
   git push origin feature/my-contribution
   ```
6. **פתח Pull Request** ותאר מה הוספת/שינית

> אנא ודא שהתוכן שלך ברור, מסודר ועומד בסגנון הקיים בפרויקט.
