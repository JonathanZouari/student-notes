## 🚀 כיצד להצטרף?

אנחנו מברכים כל תרומה! אם אתה רוצה להיות חלק מהצוות, בצע את הצעדים הבאים:

### צעדים בסיסיים:

1. **Fork את הריפוזיטורי**
   - לחץ על כפתור Fork בעמוד GitHub

2. **Clone עותק מקומי**
   ```bash
   git clone https://github.com/your-username/student-notes.git
   cd student-notes
   ```

3. **צור Branch חדש**
   ```bash
   git checkout -b feature/תיאור-התיקיה
   ```

4. **בצע את השינויים שלך**
   - עדכן קבצים קיימים או הוסף חדשים

5. **Commit עם הודעה ברורה**
   ```bash
   git commit -m "הוסף/שפר: תיאור קצר של השינוי"
   ```

6. **Push ל-Branch שלך**
   ```bash
   git push origin feature/תיאור-התיקיה
   ```

7. **פתח Pull Request**
   - בקש מהמתחזקים לבדוק ולמזג את השינויים שלך
