## כיצד להצטרף?

אנחנו שמחים לקבל תרומות מכל אחד! להלן צעדים פשוטים להתחיל:

### 1️⃣ **הכנה ראשונית**
- Fork את הריפוזיטורי לחשבונך שלך
- Clone את ה-fork שלך למחשב שלך:
  ```bash
  git clone https://github.com/YOUR_USERNAME/student-notes.git
  ```

### 2️⃣ **יצירת Branch**
- בצע branch חדש לתוכנית שלך:
  ```bash
  git checkout -b feature/your-feature-name
  ```

### 3️⃣ **ביצוע השינויים**
- בצע שינויים קטנים וספציפיים
- תוך כדי עבודה, כתוב commit messages ברורים:
  ```bash
  git commit -m "Add: description of your changes"
  ```

### 4️⃣ **שליחה ופתיחת Pull Request**
- דחוף את ה-branch שלך:
  ```bash
  git push origin feature/your-feature-name
  ```
- פתח Pull Request מהממשק של GitHub
- מלא את תבנית ה-PR בפרטים על התוכנית שלך

### 5️⃣ **ביקורת וסיום**
- התן למתחזקים לבדוק את ה-PR שלך
- השב על הערות או שאלות
- לאחר אישור - המשך ו-Merge! 🎉
