## 🚀 כיצד להצטרף?

אנחנו שמחים לקבל תרומות מכולם! הנה איך להתחיל:

### שלב 1 — Fork & Clone
```bash
# עשו Fork לפרויקט דרך GitHub
git clone https://github.com/YOUR-USERNAME/student-notes.git
cd student-notes
```

### שלב 2 — צרו Branch חדש
```bash
git checkout -b feature/your-feature-name
```

### שלב 3 — בצעו את השינויים ו-Commit
```bash
git add .
git commit -m "✨ הוספתי: תיאור קצר של השינוי"
```

### שלב 4 — פתחו Pull Request
1. דחפו את ה-Branch שלכם: `git push origin feature/your-feature-name`
2. פתחו **Pull Request** ב-GitHub
3. תארו בבירור מה השתנה ולמה
4. המתינו לסקירה — אנחנו עונים בדרך כלל תוך 48 שעות 🕐

> 💡 **טיפ:** לפני שמתחילים, מומלץ לפתוח **Issue** ולתאם עם הצוות כדי שלא נדרוס עבודה אחד של השני.
